What is HTML?

Hypertext Markup Language(HTML) is the standart markup language for creating web pages and web applications.

Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages.

HTML elements aree the duilding blocks of HTML pages. With HTML constructs images and other objects such as interactive froms may be embedded into the rendered page. HTML provides a means to create structored documents by denoting structural samantics for text such as headings a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quoters and onther items.

Markup

HTML markup consists of several key components, including those called tags (and their attributes), character based data types, character references and entity references. HTML tags most commonly come in pair like <h1> and <h1>, although some represent empty elements and so are unpaired, for example <img>.

Another important component in the HTML document type declaration, which triggers standarts mode rendering.

The following is an example of the classic "Hello, World!" program:

<!DOCTYPE html>
<html>
<head>
<title>This is a title</title>
</head>
<body>
<p>Hello world!</p>
</body>
</html>
